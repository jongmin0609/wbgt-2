# 온열 위험도 중앙 관제 대시보드 2차 개선

여러 작업자가 같은 네트워크 안의 중앙 Streamlit 서버에 접속해 심박수와 WBGT를
입력하고, 중앙서버가 작업자별 고정 프로필을 기준으로 위험도를 계산하는
다중 작업자용 대시보드입니다.

## 핵심 구조

```text
작업자 휴대폰/PC 브라우저
→ 중앙 Streamlit 서버
→ SQLite DB 저장
→ 작업자별 위험도 계산
→ 개인 대시보드와 관리자 대시보드에 반영
```

`worker_store.py`가 SQLite 저장소를 담당합니다. 기본 DB 파일은
`data/workers.db`이며, 처음 실행할 때 기본 작업자 5명이 자동 등록됩니다.

## 실행

```powershell
python -m pip install -r requirements.txt
python -m streamlit run main.py
```

또는 다음 스크립트로 중앙서버를 실행합니다.

```powershell
powershell -NoProfile -ExecutionPolicy Bypass -File .\start_servers.ps1
```

`.streamlit/config.toml`이 서버를 `0.0.0.0`에 바인딩하므로 같은 네트워크의
휴대폰에서는 PC의 로컬 IP와 Streamlit 포트로 접속할 수 있습니다.

## 접속 주소

```text
작업자 선택/입력: http://서버IP:8501
작업자 직접 링크: http://서버IP:8501?view=worker&worker_id=W001
관리자 현황:     http://서버IP:8501?view=manager
```

작업자는 처음 접속할 때 본인 이름과 식별번호를 선택합니다. 선택 후 주소창에는
`?view=worker&worker_id=W001` 같은 작업자 직접 링크가 표시됩니다. 화면이
자동으로 관리자/작업자 사이를 오가지 않도록 브라우저 저장공간 기반 자동 이동은
사용하지 않습니다.

Render로 배포하면 Wi-Fi IP 대신 공개 HTTPS 링크를 사용할 수 있습니다.
배포 절차와 작업자/관리자 링크 형식은 `DEPLOY_RENDER.md`를 참고하세요.

## 기본 작업자

```text
W001 김철수
W002 홍길동
W003 이영희
W004 박민수
W005 최지훈
```

관리자 화면의 `작업자 프로필 관리`에서 이름, 나이, 체중, 성별, 순화 판정 입력값,
활성 여부를 수정할 수 있습니다. 새 작업자도 같은 화면에서 추가할 수 있습니다.

## 작업자 사용 순서

1. 휴대폰 또는 PC에서 `http://서버IP:8501`에 접속합니다.
2. 본인의 식별번호와 이름을 선택하고 `이 작업자로 시작`을 누릅니다.
3. 심박수와 WBGT를 입력한 뒤 `중앙서버로 전송`을 누릅니다.
4. 개인 대시보드에서 위험도, 작업강도, VO2, 칼로리, 권장 휴식을 확인합니다.
5. `알림 활성화`를 한 번 누르면 해당 브라우저에서 `경고` 이상 단계에 소리와
   진동 알림을 시도합니다.

## 관리자 사용 순서

1. PC에서 `http://서버IP:8501?view=manager`에 접속합니다.
2. 전체 작업자의 최신 위험도, 심박수, WBGT, 작업강도, 대사율을 확인합니다.
3. `경고` 이상 작업자 수와 갱신 지연/미측정 작업자를 요약 카드에서 확인합니다.
4. 필요한 경우 `작업자 프로필 관리`를 열어 고정 프로필을 수정합니다.
5. 작업자 수가 늘어나면 `새 작업자 추가` 탭에서 식별번호와 이름을 등록합니다.

## 계산 흐름

1. 작업자가 심박수와 WBGT를 입력합니다.
2. 중앙 DB의 작업자 고정 프로필을 읽습니다.
3. HRR 기반 VO2 추정값을 계산합니다.
4. Keytel 계열 심박수 기반 kcal/min을 계산합니다.
5. kcal/min을 W 단위 대사율로 변환해 작업강도를 분류합니다.
6. 순화 여부에 따라 NIOSH REL/RAL 기준 WBGT 한계값을 계산합니다.
7. 실제 WBGT와 기준 WBGT의 차이로 `안전`, `주의`, `경고`, `위험`을 표시합니다.

## 참고

이 앱은 캡스톤디자인 발표와 기능 검증용 참고 시스템입니다. 현장 대응은 작업자
증상, 민감군 여부, 실제 보호구, 공식 폭염 안내와 함께 판단해야 합니다.
